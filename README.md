# 赛马分类规则

## 概述

本文件记录 Group 1/2/3 的分类判断规则。

---

## 名词解释

### 俱乐部（马主）

| 日语名称 | 通称 | 说明 |
|----------|------|------|
| `シルクレーシング` | シルク / 丝绸会 | Silk Racing |
| `キャロットファーム` | carrot / 胡萝卜 | Carrot Farm |
| `サンデーレーシング` | SR / Sunday Racing | Sunday Racing |

> 上述三家统称为**社台系俱乐部**，属于北方牧场系育成。

### 牧场

| 日语名称 | 通称 | 育成设施 | 与北方牧场系关系 |
|----------|------|----------|------------------|
| `ノーザンファーム` | NF / 北方牧场 | ✅ 有 | 北方牧场系核心 |
| `社台ファーム` | 社台牧场 | ✅ 有 | 属于社台系牧场 |
| `社台コーポレーション白老ファーム` | 白老牧场 | ❌ 无 | 特例，需看价格 |

---

## 核心原则

### 原则1：拍卖价格优先

**当一匹马同时有募集价格和拍卖价格时，只看拍卖价格判断分类**

### 原则2：G1-1 最高优先级

**生産者 = 北方牧场 时，直接判定 Group 1，不继续其他规则**

---

## Group 1 规则（按优先级）

### G1-1：北方牧场产出马 【最高优先级】

**条件**：生産者（生产牧场）为北方牧场

**数据来源**：`umas.breeder_name`

**判断逻辑**：
```
IF 生産者 == "ノーザンファーム"
   THEN Group = 1 (直接结束)
```

**说明**：北方牧场出产的马素质普遍较高，无论马主是谁、是否俱乐部募集。

---

### G1-2：高价精选/Mix拍卖马

**条件**：通过精选拍卖或 Mix Sales 购入，且税前价格 ≥ 1.5亿日元

**数据来源**：
- Select Sale：`own.netkeiba.com` 爬取，字段为 ` Select Sale`
- Mix Sales：`mix2024.csv` 的 `市場取引価格`

**判断逻辑**：
```
IF (拍卖类型 == " Select Sale" OR 拍卖类型 == " Mix Sales")
   AND (税前价格 >= 15000万)
   THEN Group = 1
```

**说明**：此规则适用于所有马。

---

### G1-3：社台系俱乐部募集（非白老牧场）

**条件**：由社台系俱乐部募集，且生产牧场不是白老牧场

**数据来源**：
- 马主：`umas.banusi_name`
- 生产牧场：`umas.breeder_name`

**判断逻辑**：
```
IF (马主 == "シルクレーシング"
    OR 马主 == "キャロットファーム"
    OR 马主 == "サンデーレーシング")
   AND (生産者 != "社台コーポレーション白老ファーム")
   THEN Group = 1
```

**说明**：社台系俱乐部（丝绸会/胡萝卜/SR）的育成属于北方牧场系。

---

### G1-4：社台牧场 × 社台系俱乐部

**条件**：社台牧场生产 + 丝绸会/胡萝卜/SR 募集

**数据来源**：
- 生产牧场：`umas.breeder_name`
- 马主：`umas.banusi_name`

**判断逻辑**：
```
IF 生産者 == "社台ファーム"
   AND (马主 == "シルクレーシング"
        OR 马主 == "キャロットファーム"
        OR 马主 == "サンデーレーシング")
   THEN Group = 1
```

**说明**：社台牧场的马在社台系俱乐部募集时，育成属于北方牧场系。

---

### G1-5：社台牧场高募集价马（非社台系俱乐部）

**条件**：社台牧场生产 + 非社台系俱乐部募集 + 募集总价 ≥ 6000万日元

**数据来源**：
- 生产牧场：`umas.breeder_name`
- 马主：`umas.banusi_name`
- 募集价格：`own.netkeiba.com` 的 `価格`

**判断逻辑**：
```
IF 生産者 == "社台ファーム"
   AND (马主 != "シルクレーシング"
        AND 马主 != "キャロットファーム"
        AND 马主 != "サンデーレーシング")
   AND (募集总价 >= 6000万)
   THEN Group = 1
```

**说明**：社台牧场的马在非社台系俱乐部募集时，若募集总价高，说明俱乐部对其评价高。

---

### G1-6：白老牧场高募集价马

**条件**：白老牧场生产 + 俱乐部募集 + 募集总价 ≥ 6000万日元

**数据来源**：
- 生产牧场：`umas.breeder_name`
- 募集价格：`own.netkeiba.com` 的 `価格`

**判断逻辑**：
```
IF 生産者 == "社台コーポレーション白老ファーム"
   AND (募集总价 >= 6000万)
   THEN Group = 1
```

**说明**：白老牧场没有育成设施，属于特例。当募集总价高时，说明俱乐部对其评价高。

---

## Group 2 规则

### G2-1：社台系牧场低价马

**条件**：社台系牧场生产 + 不满足任何 Group 1 条件

**数据来源**：
- 生产牧场：`umas.breeder_name`
- 马主：`umas.banusi_name`
- 募集价格：`own.netkeiba.com` 的 `価格`
- 拍卖价格：`own.netkeiba.com` 的 `落札価格`

**详细分类**：

| 生産者 | 俱乐部 | 募集总价 | 拍卖价 | Group |
|--------|--------|----------|--------|-------|
| 社台牧场 | 丝绸会/胡萝卜/SR | 任意 | - | **1** (G1-4) |
| 社台牧场 | 其他 | >= 6000万 | - | **1** (G1-5) |
| 社台牧场 | 其他 | < 6000万 | < 7000万或无 | **2** ← 默认 |
| 社台牧场 | 其他 | 无 | < 7000万 | **2** ← 非募集马 |
| 白老牧场 | 任意 | >= 6000万 | - | **1** (G1-6) |
| 白老牧场 | 任意 | < 6000万 | < 7000万或无 | **2** ← 默认 |
| 白老牧场 | 任意 | 无 | < 7000万 | **2** ← 非募集马 |

**说明**：社台牧场和白老牧场的非募集马（即没有募集价格），如果不满足G2-2的拍卖价格条件（7000万～1.5亿），仍然算作 Group 2。

---

### G2-2：拍卖低价区间马

**条件**：有拍卖记录 + 非北方牧场产出 + 税前价格 7000万～1.5亿日元

**数据来源**：own.netkeiba.com 的 `落札価格`

**判断逻辑**：
```
IF 有拍卖记录
   AND 生産者 != "ノーザンファーム"
   AND 拍卖类型 == "所有拍卖会"
   AND 7000万 <= 税前价格 < 1.5亿
   THEN Group = 2
```

**说明**：
- 此规则适用于所有类型的拍卖会（不限于 Select Sale / Mix Sales）
- 仅当没有命中任何 G1 条件时适用

---

### G2-3：其他牧场高募集价马

**条件**：其他牧场生产 + 非社台系俱乐部募集 + 募集总价 ≥ 6000万日元

**数据来源**：
- 募集价格：`own.netkeiba.com` 的 `価格`

**判断逻辑**：
```
IF 生産者 NOT IN ["ノーザンファーム", "社台ファーム", "社台コーポレーション白老ファーム"]
   AND 马主 NOT IN ["シルクレーシング", "キャロットファーム", "サンデーレーシング"]
   AND (募集总价 >= 6000万)
   THEN Group = 2
```

---

## Group 3 规则（默认）

不满足任何 Group 1 或 Group 2 条件的马 → **Group 3**

| 生産者 | 俱乐部 | 拍卖价 | 募集价 | Group |
|--------|--------|--------|--------|-------|
| 其他牧场 | 无/其他 | < 7000万 | < 6000万 | **3** |

## 特殊说明

1. **税率换算**：所有价格均为含税价，分类使用税前价格 = 税后价格 ÷ 1.1
2. **价格类型**：
   - 拍卖价格（落札価格）：own.netkeiba.com 爬取
   - 募集价格（価格）：own.netkeiba.com 爬取
   - Mix Sales 价格：mix2024.csv 获取
3. **拍卖场**：Select Sale 和 Mix Sales 是 G1-2 的条件之一；所有拍卖会影响 G2-2

# 2.0版本：赛马选择游戏规则

## 概述

群友每人选择10条马，系统验证是否符合以下规则。

---

## 基础规则

### 规则1：数量限制

- 必须选择 **10条** 马

### 规则2：性别比例

- **牡**（雄） + **牝**（雌） = 10
- 允许的比例：
  - 5牡 5牝
  - 4牡 6牝
  - 6牡 4牝

### 规则3：同父限制

- 同一 father 的产驹 **最多2条**
- 超过2条需要触发特殊规则1

### 规则4：Group数量限制

| 限制项 | 上限 |
|--------|------|
| Group 1 | ≤ 5条 |
| Group 1 + Group 2 | ≤ 7条 |
| Group 3 | ≥ 3条 |

### 规则5：供用年数要求

- 必须包含至少 **1条** father 的供用年数是「**1年目**」的产驹

### 规则6：海外种马要求

- 必须包含至少 **1条** father 是**海外种马**的产驹

**附加规则**：海外种马产驹与供用年数1年目产驹必须是**不同的选项**（不同的马）

> 注：两者可以是同一个 father，但必须是两条不同的产驹

---

## 特殊规则

### 特殊规则1：同父限制扩展

**触发条件**：放弃 Group 1 名额

**效果**：
- 每放弃1个G1 → 指定一条 father 的选取上限 **+1**
- 最多触发 **2次**

| 放弃G1数量 | 指定father可选数量 |
|------------|-------------------|
| 0 | ≤ 2条 |
| 1 | ≤ 3条 |
| 2 | ≤ 4条 |

### 特殊规则2：种费500万以下名额扩展

**触发条件**：选择至少 **1条** 种费低于 **500万** 日元的种马产驹

**效果**：
- 在 **Group 1** 中额外增加 **1个**「种费 < 500万」的名额
- 只能触发 **1次**

| 状态 | G1上限 |
|------|--------|
| 未触发 | 5条 |
| 已触发 | 6条（含1条种费<500万）|

**说明**：
- 本质：100换两张50
- 大多数人会用此规则换取 G1 名额，而非 G2

---

## 联合效应示例

### 示例1：同时触发两个特殊规则

- 放弃2个G1 → 某father可选4条
- 选择1条种费<500万 → G1上限变为6条
- 结果：G1 4条 + G2 3条 + G3 3条 = 10条 ✅

### 示例2：常规选择

- G1 5条（含1条种费<500万触发规则2）
- G2 2条
- G3 3条
- 含1条海外种马
- 含1条供用年数1年目
- 同一father ≤ 2条
- ✅ 符合规则
